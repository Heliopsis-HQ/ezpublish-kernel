string  $aliasKey= ''

A internal CRC32 value which is used when an alias is created.

This value is based on the filters that were used (parameters included) and is checked when an alias is accessed.

If this values differs from the configured filters (in image.ini or an override), the system will recreate the alias.

bool  $isNew= ''

Will be set to TRUE the first time the alias is created, the next time (reload of the same page) it will be FALSE.

It will also be set to TRUE every time the alias is re-created due to changes in filters (see alias_key).

bool  $isValid= ''

TRUE if the alias was created properly, that means all conversion and filters were applied without problems.

It will be FALSE if the image manager is wrongly configured or if no compatible image converters could be found.
