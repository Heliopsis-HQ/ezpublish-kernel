    Criterion description function.

    Returns the combination of the Criterion's supported operator/value, as an array of eZ\Publish\SPI\Persistence\Content\Query\Criterion\Operator\Specifications objects - Operator is one supported Operator, as an Operator::* constant - ValueType is the type of input value this operator requires, either array or single - SupportedTypes is an array of types the operator will accept - ValueCountLimitation is an integer saying how many values are expected.

    // IN and EQ are supported
return array(
    // The EQ operator expects a single value, either as an integer or a string
    new Specifications(
        Operator::EQ,
        Specifications::INPUT_TYPE_SINGLE,
        array( Specifications::INPUT_VALUE_INTEGER, Specifications::INPUT_VALUE_STRING ),
    ),
    // The IN operator expects an array of values, of either integers or strings
    new Specifications(
        Operator::IN,
        Specifications::INPUT_TYPE_ARRAY,
