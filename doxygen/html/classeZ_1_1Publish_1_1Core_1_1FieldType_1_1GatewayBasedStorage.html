Detailed Description

Storage gateway base class to be used by FieldType storages

This class gives a common basis to realized gateway based storage dispatching. It is intended to deal as a base class for FieldType storages, giving a common infrastructure to handle multiple gateways, based on the context provided by the SPI.

The method getGateway() is used in derived classes to retrieve the correct gateway implementation, based on the context. The method getPersistenceHandler() can be used to retrieve the SPI persistence handler. It is encouraged, to use this object to retrieve any eZ Publish internal data objects, in order to allow caching.
