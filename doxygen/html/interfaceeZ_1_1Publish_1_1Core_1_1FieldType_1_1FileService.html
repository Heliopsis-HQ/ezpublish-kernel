Removes the path identified by $storageIdentifier, potentially $recursive.

Attemts to removed the path identified by $storageIdentifier. If $storageIdentifier is a directory which is not empty and $recursive is set to false, an exception is thrown. Attemting to remove a non existing $storageIdentifier is silently ignored.
