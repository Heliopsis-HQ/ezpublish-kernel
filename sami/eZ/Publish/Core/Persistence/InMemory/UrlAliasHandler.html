public void publishUrlAliasForLocation(mixed $locationId, mixed $parentLocationId, string $name, string $languageCode, boolean $alwaysAvailable = false)

This method creates or updates an urlalias from a new or changed content name in a language (if published).

It also can be used to create an alias for a new location of content.
On update the old alias is linked to the new one (i.e. a history alias is generated).

$alwaysAvailable controls whether the url alias is accessible in all
languages.

Parameters

mixed $locationId
mixed $parentLocationId In case of empty( $parentLocationId ), threat as root
string $name the new name computed by the name schema or url alias schema
string $languageCode
boolean $alwaysAvailable

Return Value

void Does not return the UrlAlias created / updated with type URLAlias::LOCATION

the createCustomUrlAlias method has to be used.
