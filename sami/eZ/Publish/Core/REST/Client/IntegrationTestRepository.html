public boolean canUser(string $module, string $function, ValueObject $object, ValueObject $target = null)

Indicates if the current user is allowed to perform an action given by the function on the given objects

Example: canUser( 'content', 'edit', $content, $location );
This will check edit permission on content given the specific location, if skipped if will check on all
locations.

Example2: canUser( 'section', 'assign', $content, $section );
Check if user has access to assign $content to $section.

Parameters

string $module The module, aka controller identifier to check permissions on
string $function The function, aka the controller action to check permissions on
ValueObject $object The object to check if the user has access to
ValueObject $target The location, parent or "assignment" value object

Return Value

boolean

Exceptions

InvalidArgumentException If any of the arguments are invalid
BadStateException If value of the LimitationValue is unsupported
