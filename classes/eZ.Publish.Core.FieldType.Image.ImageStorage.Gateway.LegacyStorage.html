Abstract base class for storage gateways

This base class must be extended by storage gateways to be used with a eZ\Publish\Core\FieldType\GatewayBasedStorage based storage implementation.

The setConnection() method is called by the GatewayBasedStorage to set the connection from the current persistence context.
