List of optional configuration flags: The flag 'defer_type_update' defines if content types should be published immediately (false), when the {@link \eZ\Publish\SPI\Persistence\Content\Type\Handler::publish()} method is called, or if a background process should be triggered (true), which is then executed by the old eZ Publish core.
