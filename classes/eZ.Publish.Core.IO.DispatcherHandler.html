$alternativeHandlers

array

Structure of handlers that follows the following format: array( array( 'handler' => Handler, .. ), .. ) ie: array( array( 'handler' => $handler1, // match conditions: 'prefix' => 'var/original/', 'suffix' => '.gif,.jpg', 'contains' => 'image-versioned' ), (...) )

Exceptions

\eZ\Publish\API\Repository\Exceptions\InvalidArgumentException If $config does not contain default handler that implements Handler, handlers is unset or empty (hence you could have used default directly), one of the 'patterns' is unset or empty (hence it could have been default) or a 'handler' item does not implement Handler
