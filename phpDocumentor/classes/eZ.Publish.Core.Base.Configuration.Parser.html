Returns

arrayA plain array structure of configuration data where array clearing is marked with {@link Configuration::TEMP_INI_UNSET_VAR} and php variables are plain php values(numbers, floats, true and false). In addition strings are rtrimmed to avoid common user mistakes when dealing with configuration data (trailing whitespace). eg (ini example): [section] list[] list[]=item list[]=false var=true num=2 float=1.2 string=1,5 Result: array( 'section' => array( 'list' => array( '__UNSET__', 'item', false, ), 'var' => true, 'num' => 2, 'float' => 1.2, 'string' => '1,5', ) )
