    Assigns role to a user or user group with given limitations

    assignRole(mixed $contentId, mixed $roleId, array $limitation) 

    The limitation array looks like:

     array(
     'Subtree' => array(
         '/1/2/',
         '/1/4/',
     ),
     'Foo' => array( 'Bar' ),
     …
 )

    Where the keys are the limitation identifiers, and the respective values are an array of limitation values. The limitation parameter is optional.

    Parameters

    $contentId

    mixed

    The groupId or userId to assign the role to.

    $roleId

    mixed

    $limitation

    array

    Exceptions

    \eZ\Publish\API\Repository\Exceptions\NotFoundException If group or role is not found
    \eZ\Publish\API\Repository\Exceptions\NotFoundException If group is not of user_group Content Type
